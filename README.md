# LocalBites

A local restaurant discovery website. Browse restaurants, view menus, manage favorites, and place mock orders.

**This is a test/demo project for [FOUX](https://github.com/fouxproject).** It exists to provide a realistic web app for testing the FOUX Chrome DevTools extension, which enables visual design iteration on live sites. Generated in a single shot by Claude.

## Stack

- React + TypeScript
- Vite
- CSS Modules
- React Router v6

**Code style**: function declarations, named exports, `interface` for types, `useState` throughout, `handle` prefix event handlers, ternary conditionals.

## Running

```
npm install
npm run dev
```
